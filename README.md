# Материалы отчета лаборатории в системе [Quarto](https://quarto.org/)

Стиливой файл для оформления литературы был взят [отсюда](https://github.com/NazarovKI/gostdown/blob/master/gost-r-7-0-5-2008-numeric-iaa.csl) (спасибо Никите Платонову за наводку). Литература собирается в виде [таблицы `.csv`](https://github.com/LingConLab/2024_lab_anual_report/blob/master/bibliography.tsv), важно отмечать русские работы в столбце `note`. Кроме того стандартная болезнь: нужно выделять большие буквых в полях `TITLE` и `BOOKTITLE` фигурными скобками.

По всем вопросам пишите в issues или [Г. Морозу](https://github.com/agricolamz).
